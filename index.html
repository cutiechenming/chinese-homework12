第1题：根据拼音写汉字
4分

根据拼音在田字格内写出相应的汉字。

1. 根深dì固 ______

2. 心chí神往 ______

3. hán养性情 ______

4. 陋室míng ______

参考答案
1. 蒂 2. 驰 3. 涵 4. 铭
解题思路

这道题考查根据拼音写汉字的能力，需要结合上下文语境来推断正确的汉字。

1. 根深dì固：这是一个成语，意思是基础深厚，不容易动摇，应写作"根深蒂固"。

2. 心chí神往：心神奔向所向往的事物，形容一心向往，应写作"心驰神往"。

3. hán养性情：指滋润养育，使性格、品格得到培养，应写作"涵养性情"。

4. 陋室míng：指刘禹锡的《陋室铭》，应写作"陋室铭"。

第2题：词语和标点选择
3分

填入【甲】【乙】两处的词语和标点，最恰当的一项是（ ）

A．【甲】墨守 【乙】《聊斋》
B．【甲】恪守 【乙】 "聊斋 "
C．【甲】恪守 【乙】《聊斋》
D．【甲】墨守 【乙】 "聊斋 "
参考答案
B
解题思路

墨守：指固执保守，不思进取。

恪守：指谨慎而恭顺地遵守、严格遵守。

结合"【甲】君子之道，于书香浸润中寻得一处精神的世外桃源"可知，甲处所填词语是用于形容严格遵守君子之道，故应选填"恪守"。

结合"书房可以闹，蒲松龄设一茶棚，人来人往中记录奇闻异事，【乙】之名由此而来"可知，乙处要填的是蒲松龄的书房的名字，故应填"聊斋"；"聊斋"是特指蒲松龄的书房，故应给它加上双引号。

古诗赏析与解题思路

《谷口书斋寄杨补阙》逐句赏析

泉壑带茅茨，云霞生薜帷。
山泉沟壑环绕着我的茅屋书斋，云霞从爬满薜荔的墙帷升起。
竹怜新雨后，山爱夕阳时。
新雨过后的翠竹分外可爱，夕阳映照下的山色更显迷人。
闲鹭栖常早，秋花落更迟。
悠闲的白鹭早早栖息，秋日的花朵迟迟不落。
家僮扫萝径，昨与故人期。
家童清扫着布满松萝的小径，昨天已与老朋友约定好相见。

第5题解析：

C选项错误。颈联"闲鹭栖常早，秋花落更迟"意为：悠闲的白鹭早早就栖息，秋日的花朵迟迟不凋谢。"闲鹭"早早栖息，体现出一种闲适；"秋花"凋落较迟，展现出秋景的美好和生命力。这里是在描绘山谷的宁静祥和、富有生机，而不是山里生活无聊。

第6题解析：

1. 前三联写景：通过对书斋周围泉壑、云霞、竹、山、鹭、花等自然景色的描写，展现了幽静清新的美景，字里行间透露出诗人对这一环境的喜爱与陶醉，突出诗人欣喜愉悦的心情。

2. 尾联叙事抒情：诗人特意让家僮打扫布满松萝的小径，点明了昨天已经与友人约定好来访。诗人安排家僮打扫路径，这一行为体现了他对友人到来的重视，通过描述与友人的约定，进一步强化了这种情感。尾联突出表现诗人对友人来访的期盼（或邀约的诚意盛情）。

第9题解析：

根据文言文断句的方法，先梳理句子大意，分清层次，然后断句，反复诵读加以验证。主语和谓语之间，谓语和宾语、补语之间一般要作停顿。

"他日忽自悔谋立誓戒之"：有一天我忽然自己悔悟了，打算发誓戒掉它。

"他日"是时间状语，"忽自悔"是原因，"谋立誓戒之"是举动，故中间分别断开。

第11题解析：

选A角度：漫画式夸张描写

作者运用漫画式夸张描写，将"书痴"醉心读书的形象刻画得入木三分，极具喜剧效果，同时又让人真切感受到他对书的痴迷，心生敬意。如"甫晨起，即科头，随意抽书一帙，据坐批阅之。顷至会心处，或歌或叹，或笑或泣，或怒骂，或大叫称快，或咄咄诧异，或卧而思，起而狂走"，晨起不戴帽子就读书，读到精彩处，情绪全然外放，或歌或泣、或怒或喜，甚至起身狂走，这些夸张的行为动作，生动地展现了主人公沉浸书中世界不能自拔的状态。

选B角度：侧面烘托

通过侧面烘托的手法，从家人和访客的反应来展现主人公的书痴形象，使人物形象更加丰满真实。家人看到他读书时的种种癫狂举动，"悉骇愕，罔测所指。乃窃相议，俟稍定，始散去"，从家人的惊愕、猜测以及悄悄议论、等待他平静后才敢散去，侧面反映出主人公读书时的忘情投入。

第12题解析：

本题考查对比阅读和文化探究能力，需要联系三篇课文，分析古代文人"醉"与"痴"背后的精神追求：

  • 《醉书斋记》：郑日奎沉醉于读书，在书中寻找精神寄托
  • 《湖心亭看雪》：张岱痴迷于雪景，追求超凡脱俗的境界
  • 《醉翁亭记》：欧阳修陶醉于山水，追求与民同乐的境界

共同点：这些文人都在自己的雅趣中陶冶性情，追求精神自由，保持独立人格，不随波逐流。

第13题：名著阅读《儒林外史》人名深意
6分

文学作品中人物的姓名往往能体现出作者的匠心。姓名或体现了人物性格，或暗示了人物命运，或寄寓了作者态度，或暗示了文章主旨。请参考示例（不要求仿照），说说《儒林外史》中范进的人名有哪些深意。

示例：林冲，冲含有"冲击、冲锋"之意，由于遭到高俅等权贵的陷害，林冲被迫害至家破人亡的境地。先是妻子受辱，接着自己也被刺配沧州，这一系列的打击使得林冲的性格开始发生变化。火烧草料场是重要的转折点，他从原来的逆来顺受、委曲求全转变为在绝境中展现出血性和反抗精神。"冲"字暗示了他直率刚烈的性格以及在遭受一系列冤屈后最终走上反抗道路的命运走向，很好地诠释了他从逆来顺受的八十万禁军教头转变为梁山英雄的过程。

参考答案

示例1：范进，"范"有模范、规范之意，"进"即进取、晋升， 范进对功名利禄有着近乎疯狂的追求，他将自己的人生几乎全部押注在科举考试之上，尽管多年屡试不中，仍矢志不渝，即便生活陷入极度贫困，甚至不惜让家人忍饥挨饿，也要继续参加科举考试。 他性格软弱且自卑，在未中举之前备受邻里嘲笑和岳父胡屠户的羞辱，但始终坚守自己的科举之路，直到晚年终于中举，却又因过于激动而发疯，尽显丑态。 "范""进"二字尖锐地讽刺了他对功名之追求的病态执着，批判了封建科举制度对士人精神世界的扭曲和戕害。

示例2：范进，"范"含"范式、范例"之意，"进"指科举进阶、功名递进。 他是封建科举制度下无数士人的缩影，将科举视为人生唯一正途，从青年到暮年，数十年如一日沉溺考场，即便穷困潦倒、遭人唾弃也不愿回头。 中举前他唯唯诺诺、忍辱负重，中举后却狂喜失态、疯癫发狂，前后判若两人。 "范进"二字既点出他是科举制度塑造的"标准范本"，又暗讽这种"进阶"之路对人性的异化，深刻揭露了封建科举对士人灵魂的腐蚀与摧残。

参考答案

15. ①聊慰乡情，甜到心底 ②全无味道，百思不解

16. 命名示例：解忧"华莱士"
镜头描写示例：屋外尘暴蔽日，白天烈日烤人（或：晚上夜凉如水）； 屋内"华莱士"瓜外表金黄、瓜瓤白中带绿如翡翠，吃瓜人正在吃瓜，津津有味，如吃蜂蜜一般。

17. ①第一个"瓜"指的是"看热闹的人"爱看的"热闹"， 第二个"瓜"指的是地里生长的一种水果"真瓜"。
②运用动作描写（或：细节描写）的方法， 通过"坐""吃""甩掉""捧过""吸"等一系列动词， 生动细致地描写了在冬腊月吸食冬瓜的情景，表现了作者对往昔生活的留恋和怀念之情。

18. 示例：对瓜来说，离地一天，味减一半，人们最爱恋的味道最好的瓜应该是刚从地里摘下来的； 对世间万物而言，世界瞬息万变，人生的许多美好只能有一次，过后唯有存在记忆里。

19. 示例：《社戏》叙述了"我"少年时一段看水乡社戏的往事，主要描写了"月夜行船""赵庄看戏""停船吃豆"等情节，表现了作者对童年美好生活的回忆和留恋的心情； 本文主要记叙了作者三次吃瓜的不同经历和感受，抒发了作者对往昔生活的回忆和留恋之情。 因此，鲁迅"我实在再没有吃到那夜似的好豆，一一也不再看到那夜似的好戏"、梁衡"而今吃瓜高楼上，淡而无味也"两人产生相似感受的原因是：往昔的生活快乐新奇，是最美好的回忆； 两人都对童年的生活充满了回忆和留恋之情。
